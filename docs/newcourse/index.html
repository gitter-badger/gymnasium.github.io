Use the following settings to create a new Gymnasium course in Studio.

Setting up a New Course

Default Course Settings

  • Course Name: [Full Name] e.g. Coding for Designers
  • Organization: Gymnasium
  • Course Number: [Course ID] e.g. 100
  • Course Run: 0

Note

All new courses launch with Course Run set to 0. This value increments when the course is relaunched.

Course Team

Add Catalog Content

  • Course Details
  • Course Artwork
  • Instructor Bio
  • Instructor Photo

Note

  • Refer to the Gymnasium Assets Site for content
  • The document with My Progress in the title for the course outline
  • edX concepts translated to Gymnasium
    • Section = Lesson
    • Subsection = Chapter
    • Unit = Video or Assessment

Adding Course Material

Create the Welcome Section

  • Create a new Section
    • Name: Welcome
  • Add a Subsection
    • Name: Welcome
  • Add a Unit
    • Name: Welcome
    • Type: Raw HTML
  • Add a Unit
    • Name: Test Your Knowledge (Optional)
    • Type: Problem > Multiple Choice
    • Settings
      • Problem weight: 100
      • Maximum Attempts: 1
      • Show Answer: Never

Create a Lesson

  • Create a new Section
    • Name: Lesson One: Title of this lesson

Add a Chapter

  • Create a new Subsection
    • Name: Title of this Chapter
  • Add a Unit
    • Name: Title of this Chapter
  • Add a Component:
    • Type: HTML > Raw HTML
  • Edit and replace the HTML content with the Wistia embed code for the current video

Add a Quiz

  • Create a new Subsection
    • Name: Lesson [Number] Quiz
  • Add a new Unit
    • Name: Lesson [Number] Quiz
    • Type: Problem > Multiple Choice
    • Settings
      • Problem weight: 100
      • Maximum Attempts: [Blank]
      • Show Answer: Finished

Add a Materials Section

  • Create a new subsection
    • name: Homework & Materials
  • Add a Unit
    • name: Homework & Materials
  • Add a Component: HTML > Raw HTML
<h2><a href="#">Lesson [Number] Handout</a></h2>
<p>This includes assignments, readings, and a transcript.</p>
<h2><a href="#">Lesson [Number] Files</a></h2>
<p>This includes files used in the videos and for assignments.</p>

Add the Final Exam

  • Create a new Section
    • Name: Final Exam
  • Add a Subsection
    • Name: Final Exam
  • Add a Unit
    • Name: Asssessment
    • Type: Problem > Multiple Choice
    • Settings
      • Problem weight: 100
      • Maximum Attempts: 2
      • Show Answer: Finished
